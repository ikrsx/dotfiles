# Dotfiles
A collection of configuration files for my linux workflow.

## Dofiles Index 🗂️
Below are the list of programs for whom I have created the dotfiles:
1. **Niri:** A scrollable window manager
2. **Alacritty:** Terminal emulator
3. **Fuzzel:** Lightweight application launcher
4. **Mako:** Notification daemon
5. **Hyperlock:** Lockscreen application
6. **Hyperpaper:** Wallpaper manager
7. **Waybar:** Top bar for the window manager
8. **Neovim:** Terminal based text editor
9. **Tmux:** Terminal multiplexer
10. **Yazi:** Terminal file manager
